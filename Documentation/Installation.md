# Installation

Install the module as dependency.

```sh
npm install @doomcommerce/shopify-search
```

<br>

Alias the module for ease of use.

```json
"paths" : {
    "Search" : [ "node_modules/@doomcommerce/shopify-search" ] ,
}
```
